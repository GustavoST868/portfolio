Algoritmo de Kruskal

Desenvolvi um projeto para encontrar a árvore geradora mínima de um grafo usando o algoritmo de Kruskal, um conceito fundamental em Teoria dos Grafos. O projeto utiliza as bibliotecas padrão stdio.h e stdlib.h. Criei estruturas de dados para representar arestas e grafos, incluindo a estrutura Edge para as arestas e Graph para o grafo. As funções find e join implementam a estrutura essencial para verificar e unir componentes conectados. A função Kruskal ordena as arestas por peso e constrói a árvore geradora mínima verificando componentes desconectados. A função main obtém os dados do usuário e utiliza o algoritmo de Kruskal para calcular e imprimir a árvore geradora mínima. Este projeto foi uma excelente oportunidade para entender e aplicar conceitos de grafos e algoritmos de otimização.

Estrutura de Armazenamento C#

Desenvolvi um projeto em C# que implementa um sistema de armazenamento e análise de dados. Utilizei classes para organizar funcionalidades como inserção, remoção, busca, impressão e salvamento de dados. A classe Store gerencia diferentes tipos de dados (texto, números e compostos) e possui métodos para inserção e remoção de dados, busca, impressão e salvamento em arquivo. A classe CheckType usa expressões regulares para determinar o tipo de dado. A classe Analyze realiza análises estatísticas, como média, variância e desvio padrão dos dados numéricos, além de calcular a frequência de dados textuais. O método Main no Program executa testes abrangentes para verificar a funcionalidade do sistema. Este projeto foi uma ótima oportunidade para praticar conceitos de programação orientada a objetos e análise de dados em C#.

Sistema de Lista de Tarefas

Recentemente, desenvolvi um Sistema de Listas de Tarefas como parte do meu treinamento em programação. O sistema permite criar, excluir, pesquisar e listar tarefas de maneira eficiente. Para garantir a segurança dos dados, implementei uma classe de criptografia que protege as informações das tarefas. As tarefas são armazenadas em arquivos, sendo criptografadas e descriptografadas conforme necessário. A classe TaskStorage gerencia todo o processo de armazenamento seguro e recuperação dos dados. Além disso, a classe Task facilita a manipulação das tarefas, permitindo operações como criação, exclusão e busca. Este projeto me ajudou a aprimorar minhas habilidades em manipulação de dados, segurança e desenvolvimento de software.
